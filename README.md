# 📂 LLM 연습용 데이터 저장소

이 저장소는 Large Language Model(LLM) 실습을 위해 구성된 데이터셋 모음입니다.
텍스트 분류, 요약, 정보 추출, 프롬프트 엔지니어링 등 다양한 작업을 연습할 수 있도록 설계되었습니다.
LLM을 학습하고 활용하고자 하는 개발자 · 연구자 · 학생 모두에게 유용한 자료를 제공합니다.

# 🔍 주요 특징

다양한 LLM 실습 작업 지원

Classification / Summarization / Extraction / Prompt Engineering 등

초보자도 쉽게 활용 가능한 구조화된 데이터

커스터마이징 용이 — 실험 목적에 맞게 데이터 추가·수정 가능

학습·평가·모델 비교에 활용 가능

# 📁 폴더 구조
```
/asset/       # 원본 및 가공된 텍스트 데이터

/src/    # 코딩관련 예제

/docs/       # 데이터 설명서 및 참고 문서
```

필요 시 실제 저장소 구조에 맞게 수정하세요.

# 🚀 사용 목적

LLM 기반 서비스 및 애플리케이션 개발 실습

다양한 프롬프트 전략 테스트 및 성능 비교

텍스트 데이터 전처리 및 구조화 연습

교육, 연구, 개인 프로젝트용 데이터 활용
